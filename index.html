Sobre Voice Clicker Pro

Una herramienta diseñada para optimizar el flujo de trabajo mediante el reconocimiento de voz local, permitiendo ejecutar macros, clics y comandos de teclado sin usar las manos.

Directiva de Privacidad

Última actualización: 18 de abril de 2026

1. Compromiso de Privacidad

En Sergio Labs, la privacidad de nuestros usuarios es nuestra prioridad. Voice Clicker Pro ha sido desarrollado bajo el principio de "Privacidad por Diseño". Esto significa que la aplicación está construida para funcionar sin necesidad de recolectar datos personales.

2. Recopilación y Uso de Datos

Voice Clicker Pro NO recopila, NO almacena y NO transmite ningún tipo de información personal, datos de identificación, geolocalización o historial de uso a servidores externos o terceros.

3. Uso del Micrófono y Procesamiento de Voz

La aplicación requiere acceso al micrófono para su funcionamiento principal. El procesamiento del audio se realiza íntegramente de forma local en el dispositivo del usuario mediante modelos de reconocimiento de voz offline. El audio capturado nunca sale del dispositivo ni es grabado de forma permanente.

4. Almacenamiento Local

Cualquier configuración, macro o preferencia establecida por el usuario se almacena exclusivamente en el almacenamiento local del dispositivo. Estos datos no son accesibles para Sergio Labs.

5. Seguridad y Transmisión de Datos

La aplicación no contiene rastreadores (trackers) ni herramientas de analítica de terceros. No se realizan conexiones a internet para el envío de telemetría.

6. Derechos del Usuario (GDPR)

Al no recopilar datos, el usuario mantiene el control total sobre su información. Cumplimos con el Reglamento General de Protección de Datos (GDPR) garantizando que no existe tratamiento de datos personales en nuestros sistemas.
