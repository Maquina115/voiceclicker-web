Privacy Policy – Voice Clicker Pro

1. Introducción

La presente Política de Privacidad regula el tratamiento de datos personales realizado por la aplicación de escritorio Voice Clicker Pro (en adelante, “la Aplicación”), desarrollada para sistemas Windows y orientada al control del ordenador mediante comandos de voz y automatización de teclado y ratón.

Este documento cumple con el Reglamento (UE) 2016/679 (Reglamento General de Protección de Datos, “GDPR”) y con los requisitos de privacidad exigidos por Microsoft Store.

Voice Clicker Pro ha sido diseñada bajo un enfoque de privacidad por defecto y por diseño, garantizando que todo el procesamiento se realiza de forma local en el dispositivo del usuario, sin transmisión de datos a servidores externos.

3. Datos que se recopilan

Voice Clicker Pro ha sido diseñada para no recopilar, almacenar ni transmitir datos personales. Esto incluye:

  • No se recogen nombres, direcciones, correos electrónicos ni identificadores.
  • No se crean perfiles de usuario.
  • No se registran comandos de voz.
  • No se almacenan grabaciones de audio.
  • No se recopilan métricas de uso, telemetría ni analíticas.

La ausencia de recopilación de datos constituye una decisión técnica y jurídica basada en el principio de minimización de datos (art. 5.1.c GDPR).

4. Uso del micrófono

La Aplicación requiere acceso al micrófono para poder ejecutar comandos de voz. Este acceso se utiliza exclusivamente para:

  • Capturar audio en tiempo real.
  • Procesar dicho audio localmente mediante el motor de reconocimiento de voz offline Vosk.
  • Interpretar comandos para automatizar acciones del sistema.

El audio capturado:

  • No se envía a Internet.
  • No se transmite a terceros.
  • No se almacena en el dispositivo.
  • No se utiliza para entrenar modelos.

El procesamiento es estrictamente efímero: el audio se analiza en memoria y se descarta inmediatamente tras su interpretación.

5. Procesamiento local

Toda la funcionalidad de Voice Clicker Pro opera de forma 100% local. Esto significa:

  • No existen servidores remotos.
  • No se realizan conexiones externas.
  • No se envían datos a la nube.
  • No se utilizan servicios de terceros que impliquen transferencia de datos.

Este enfoque garantiza que el usuario mantiene el control total sobre su información y que no se produce ningún tratamiento de datos personales más allá del estrictamente necesario para el funcionamiento inmediato de la Aplicación.

6. Uso de red (LAN)

La Aplicación puede, opcionalmente, comunicarse con otros dispositivos dentro de la red local (LAN) del usuario para funciones de control remoto. Esta comunicación:

  • No sale de la red local.
  • No implica transmisión de datos personales.
  • Se limita a comandos técnicos necesarios para la automatización.

7. Base legal del tratamiento (GDPR – Art. 6)

Dado que la Aplicación no recopila datos personales, la base legal aplicable se limita al tratamiento estrictamente necesario para:

  • Art. 6.1.b GDPR – Ejecución de un contrato: el procesamiento de audio en tiempo real es necesario para proporcionar la funcionalidad solicitada por el usuario.
  • Art. 6.1.f GDPR – Interés legítimo: garantizar el funcionamiento seguro y eficiente de la Aplicación.

8. Conservación de datos

Voice Clicker Pro no conserva ningún dato personal. El audio procesado se elimina inmediatamente tras su análisis en memoria. No existen registros, logs, historiales ni bases de datos asociadas al usuario.

9. Seguridad

Aunque no se almacenan datos, la Aplicación implementa medidas de seguridad para proteger el entorno de ejecución:

  • Procesamiento local sin transmisión externa.
  • Uso de librerías offline verificadas.
  • Ausencia de telemetría o canales de salida.
  • Restricción de permisos al mínimo necesario.

10. Transferencias internacionales

No se realizan transferencias internacionales de datos, ya que la Aplicación no transmite información fuera del dispositivo del usuario.

11. Servicios de terceros

La Aplicación utiliza la librería Vosk para reconocimiento de voz offline. Esta librería:

  • Opera completamente en local.
  • No envía datos a servidores externos.
  • No recopila información del usuario.

12. Derechos del usuario

Aunque la Aplicación no procesa datos personales, el usuario conserva los derechos reconocidos por el GDPR:

  • Acceso
  • Rectificación
  • Supresión
  • Limitación del tratamiento
  • Oposición
  • Portabilidad

En la práctica, estos derechos no requieren ejercicio, ya que no existe información personal que gestionar.

13. Derecho a presentar una reclamación

El usuario puede presentar una reclamación ante la autoridad de control competente en materia de protección de datos de su país de residencia dentro de la Unión Europea.

1. Introduction

This Privacy Policy governs the processing of personal data performed by the desktop application Voice Clicker Pro (“the Application”), designed for Windows systems and intended for voice‑based computer control and keyboard/mouse automation.

This document complies with the General Data Protection Regulation (EU) 2016/679 (“GDPR”) and with the privacy requirements established by Microsoft Store.

Voice Clicker Pro has been developed following a privacy‑by‑design and privacy‑by‑default approach, ensuring that all processing occurs locally on the user’s device, with no transmission of data to external servers.

3. Data Collected

Voice Clicker Pro is designed to not collect, store, or transmit any personal data. This includes:

  • No names, addresses, emails, or identifiers.
  • No user accounts or profiles.
  • No voice commands are recorded.
  • No audio is stored.
  • No usage analytics or telemetry.

This absence of data collection is a deliberate technical and legal decision aligned with the GDPR principle of data minimization (Art. 5.1.c).

4. Microphone Usage

The Application requires access to the microphone to process voice commands. This access is used exclusively to:

  • Capture real‑time audio.
  • Process audio locally using the offline Vosk engine.
  • Interpret commands for system automation.

Captured audio:

  • Is not sent to the Internet.
  • Is not transmitted to third parties.
  • Is not stored on the device.
  • Is not used to train models.

Processing is ephemeral: audio is analyzed in memory and immediately discarded.

5. Local Processing

All functionality operates 100% locally. This means:

  • No remote servers.
  • No external connections.
  • No cloud transmission.
  • No third‑party services involving data transfer.

6. Network Usage (LAN)

The Application may optionally communicate with other devices within the user’s local network (LAN) for remote‑control features. Such communication:

  • Never leaves the local network.
  • Does not involve personal data.
  • Is limited to technical automation commands.

7. Legal Basis (GDPR – Art. 6)

Since the Application does not collect personal data, the applicable legal bases relate solely to the processing required to provide functionality:

  • Art. 6.1.b – Performance of a contract: real‑time audio processing is necessary to deliver the requested features.
  • Art. 6.1.f – Legitimate interest: ensuring secure and efficient operation of the Application.

8. Data Retention

Voice Clicker Pro does not retain any personal data. Audio is processed in memory and discarded immediately. No logs, histories, or databases are created.

9. Security

Although no data is stored, the Application implements security measures such as:

  • Local‑only processing.
  • Verified offline libraries.
  • No telemetry or outbound channels.
  • Minimal permission usage.

10. International Transfers

No international data transfers occur, as no information leaves the user’s device.

11. Third‑Party Services

The Application uses the Vosk offline speech recognition library. This library:

  • Operates entirely locally.
  • Does not send data externally.
  • Does not collect user information.

12. User Rights

Users retain all GDPR rights, including:

  • Access
  • Rectification
  • Erasure
  • Restriction
  • Objection
  • Portability

In practice, these rights do not require exercise because no personal data is processed.
