Política de Privacidad

Voice Clicker Pro ha sido diseñada desde cero para no recopilar ni transmitir datos personales. Procesamiento 100% local en tu dispositivo.

🔒 Todo el audio se procesa localmente con el motor Vosk. Nada sale de tu equipo. Sin servidores, sin cloud, sin telemetría.

01 Introducción

La presente Política de Privacidad regula el tratamiento de datos personales realizado por la aplicación de escritorio Voice Clicker Pro, desarrollada para sistemas Windows y orientada al control del ordenador mediante comandos de voz y automatización de teclado y ratón.

Este documento cumple con el Reglamento (UE) 2016/679 (GDPR) y con los requisitos de privacidad exigidos por Microsoft Store. Voice Clicker Pro ha sido diseñada bajo un enfoque de privacidad por defecto y por diseño, garantizando que todo el procesamiento se realiza de forma local en el dispositivo del usuario.

Contacto: support@sergiolabs.com · 2026

03 Datos que se recopilan

Voice Clicker Pro ha sido diseñada para no recopilar, almacenar ni transmitir datos personales. Esto incluye:

  • No se recogen nombres, direcciones, correos electrónicos ni identificadores.
  • No se crean perfiles de usuario.
  • No se registran comandos de voz.
  • No se almacenan grabaciones de audio.
  • No se recopilan métricas de uso, telemetría ni analíticas.

La ausencia de recopilación de datos constituye una decisión técnica y jurídica basada en el principio de minimización de datos (art. 5.1.c GDPR).

04 Uso del micrófono

La Aplicación requiere acceso al micrófono exclusivamente para capturar audio en tiempo real, procesarlo localmente mediante el motor Vosk e interpretar comandos para automatizar acciones del sistema.

El audio capturado no se envía a Internet, no se transmite a terceros, no se almacena en el dispositivo y no se utiliza para entrenar modelos. El procesamiento es estrictamente efímero: el audio se analiza en memoria y se descarta inmediatamente.

05 Procesamiento local

Toda la funcionalidad de Voice Clicker Pro opera de forma 100% local: no existen servidores remotos, no se realizan conexiones externas, no se envían datos a la nube y no se utilizan servicios de terceros que impliquen transferencia de datos.

06 Uso de red (LAN)

La Aplicación puede, opcionalmente, comunicarse con otros dispositivos dentro de la red local (LAN) para funciones de control remoto. Esta comunicación no sale de la red local, no implica transmisión de datos personales y se limita a comandos técnicos necesarios para la automatización.

07 Base legal del tratamiento (GDPR – Art. 6)

  • Art. 6.1.b – Ejecución de un contrato: el procesamiento de audio en tiempo real es necesario para proporcionar la funcionalidad solicitada.
  • Art. 6.1.f – Interés legítimo: garantizar el funcionamiento seguro y eficiente de la Aplicación.

08 Conservación de datos

Voice Clicker Pro no conserva ningún dato personal. El audio procesado se elimina inmediatamente tras su análisis en memoria. No existen registros, logs, historiales ni bases de datos asociadas al usuario.

09 Seguridad

  • Procesamiento local sin transmisión externa.
  • Uso de librerías offline verificadas (Vosk).
  • Ausencia de telemetría o canales de salida.
  • Restricción de permisos al mínimo necesario.

10 Transferencias internacionales

No se realizan transferencias internacionales de datos, ya que la Aplicación no transmite información fuera del dispositivo del usuario.

11 Servicios de terceros

La Aplicación utiliza la librería Vosk para reconocimiento de voz offline. Esta librería opera completamente en local, no envía datos a servidores externos y no recopila información del usuario.

12 Derechos del usuario

El usuario conserva todos los derechos reconocidos por el GDPR (acceso, rectificación, supresión, limitación, oposición y portabilidad). En la práctica, estos derechos no requieren ejercicio, ya que no existe información personal que gestionar.

13 Derecho a reclamación

El usuario puede presentar una reclamación ante la autoridad de control competente en materia de protección de datos de su país de residencia dentro de la Unión Europea.

14 Cambios en esta política

Voice Clicker Pro podrá actualizar esta Política de Privacidad para reflejar mejoras técnicas, cambios regulatorios o nuevas funcionalidades. La versión vigente estará siempre disponible en el sitio oficial.

03 Data Collected

Voice Clicker Pro is designed to not collect, store, or transmit any personal data. This includes:

  • No names, addresses, emails, or identifiers.
  • No user accounts or profiles.
  • No voice commands are recorded.
  • No audio is stored.
  • No usage analytics or telemetry.

This is a deliberate technical and legal decision aligned with the GDPR principle of data minimization (Art. 5.1.c).

04 Microphone Usage

The Application requires access to the microphone exclusively to capture real-time audio, process it locally using the offline Vosk engine, and interpret commands for system automation.

Captured audio is not sent to the Internet, not transmitted to third parties, not stored on the device, and not used to train models. Processing is ephemeral: audio is analyzed in memory and immediately discarded.

05 Local Processing

All functionality operates 100% locally. There are no remote servers, no external connections, no cloud transmission, and no third-party services involving data transfer.

06 Network Usage (LAN)

The Application may optionally communicate with other devices within the user's local network (LAN) for remote-control features. Such communication never leaves the local network, does not involve personal data, and is limited to technical automation commands.

07 Legal Basis (GDPR – Art. 6)

  • Art. 6.1.b – Performance of a contract: real-time audio processing is necessary to deliver the requested features.
  • Art. 6.1.f – Legitimate interest: ensuring secure and efficient operation of the Application.

08 Data Retention

Voice Clicker Pro does not retain any personal data. Audio is processed in memory and discarded immediately. No logs, histories, or databases are created.

09 Security

  • Local-only processing with no external transmission.
  • Verified offline libraries (Vosk).
  • No telemetry or outbound channels.
  • Minimal permission usage.

10 International Transfers

No international data transfers occur, as no information leaves the user's device.

11 Third-Party Services

The Application uses the Vosk offline speech recognition library. This library operates entirely locally, does not send data externally, and does not collect user information.

12 User Rights

Users retain all GDPR rights (access, rectification, erasure, restriction, objection, portability). In practice, these rights do not require exercise because no personal data is processed.
